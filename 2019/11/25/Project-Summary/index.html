Project-Summary

我叫康锐朗，来自广东省佛山市顺德区，现在在美国佛罗里达州西棕榈滩上学。 这次我制作的app是一个AI购物清单，用于给一些容易忘记购物时要买什么的用户提供方便的，app是一个通过照相扫描，然后AI识别出这个物品，接着会加入到这个清单，清单会越积累越多，当客户需要去购物时，只需要对着自己买的东西拍下一张照片，就可以清除list上面的物品。
